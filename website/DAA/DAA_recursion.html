Analysis of Recursive Algorithms

Recursion is a technique where a function calls itself to solve smaller instances of the problem.

Key Concepts in Recursion

  • Base Case and Recursive Case
  • Recursive Functions
  • Recursion Tree Method
  • Master Theorem

Base Case

The base case stops the recursion. Without it, recursion would continue infinitely.

function factorial(n) {
    if (n === 0) return 1; // Base case
    return n * factorial(n - 1); // Recursive case
}

Recursion Tree Method

This method visualizes recursive calls as a tree structure, with each node representing a subproblem.

Master Theorem

The Master Theorem is used to determine the time complexity of recurrence relations:

Formula: T(n) = aT(n/b) + O(n^k)

It helps in solving divide-and-conquer algorithms efficiently.
